# BPMN (Business Process Model and Notation)

Los diagramas de BPMN son una herramienta visual utilizada para modelar y representar procesos de negocio.
En el caso de nuestro sistema se implementaron probisionalmente tres diagramas de BPMN para representar los siguientes procesos:

- Proceso de captacion de la persona interesada (lead).
- Proceso de reserva de cupo y pago de inscripcion.
- Proceso de notificaciones a personas interesadas que no logran reservar cupo.
