# RF-COM-05. El Bot debe proporcionar información detallada de cada evento

**Descripción:**  
El sistema debe permitir que el usuario consulte información detallada de un evento específico sea por el cual inicio el chat o por uno que le haya pedido al Bot debe incluir la duración, modalidad, precio, requisitos de ingreso y tipo de certificación otorgada.

**Historia de usuario**
*Como* usuario interesado en un evento específico
*Quiero* consultar información detallada del evento
*Para* evaluar si cumple con mis necesidades y expectativas

**Criterios de aceptación:**

- El usuario puede seleccionar un evento del listado.
- El Bot muestra todos los campos informativos definidos.
- La información se presenta de forma clara y estructurada.
