# RF-COM-06. El Bot debe informar fechas de inicio y horarios disponibles

**Descripción:**  
El Bot debe proporcionar las fechas de inicio programadas y los horarios disponibles para cada evento cuando el usuario lo solicite.

**Historia de usuario**
*Como* usuario interesado en inscribirme en un evento
*Quiero* conocer las fechas de inicio y horarios disponibles
*Para* organizar mi tiempo y confirmar mi disponibilidad

**Criterios de aceptación:**

- El usuario puede preguntar por próximas fechas o en el contexto adecuado el Bot las debe dar (el usuario pide información específica del evento sin mencionar directamente que quiere las fechas).
- El Bot responde con fechas actualizadas.
- Se muestran horarios organizados por jornada (mañana, tarde, noche, etc.).
