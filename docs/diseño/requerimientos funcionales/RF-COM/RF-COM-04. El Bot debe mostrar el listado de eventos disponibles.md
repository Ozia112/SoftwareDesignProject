# RF-COM-04. El Bot debe mostrar el listado de eventos disponibles

**Descripción:**  
El sistema debe ser capaz de visualizar los eventos disponibles, en caso de que el usuario quiera saber sobre otros eventos también debe ser capaz de mostrar al menos el nombre del evento y una breve descripción.

**Historia de usuario**
*Como* usuario interesado en conocer nuevos eventos
*Quiero* visualizar el listado actual de los eventos disponibles
*Para* conocer las opciones activas y decidir cual me interesa

**Criterios de aceptación:**

- El bot es capaz de obtener información de un evento desde la base de datos
- Si el usuario solicita al Bot este devolverá información sobre los eventos disponibles.
- El Bot muestra una lista estructurada con todos los eventos activos.
- La información proviene de una base de datos que puede ser modificada para agregar o reducir la cantidad eventos.
